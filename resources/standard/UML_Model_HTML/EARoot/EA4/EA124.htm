Defines a series of corner points, including the height of the object relative to the road reference line. For areas, the points should be listed in counter-clockwise order.<br/>An #lt;outline#gt; element shall be followed by one or more #lt;cornerRoad#gt; element or by one or more #lt;cornerLocal#gt; element.<br/><br/>OpenDRIVE 1.4 outline definitions (without #lt;outlines#gt; parent element) shall still be supported.<br/><br/>
