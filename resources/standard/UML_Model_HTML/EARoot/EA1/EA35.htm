Ancillary data should be described near the element it refers to. Ancillary data contains data that are not yet described in OpenDRIVE, or data that is needed by an application for a specific reason. Examples are different road textures.<br/>In OpenDRIVE, ancillary data is represented by #lt;userData#gt; elements. They may be stored at any element in OpenDRIVE.<br/>
