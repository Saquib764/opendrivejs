The width of a lane is defined along the t-coordinate. The width of a lane may change within a lane section.<br/>Lane width and lane border elements are mutually exclusive within the same lane group. If both width and lane border elements are present for a lane section in the OpenDRIVE file, the application must use the information from the #lt;width#gt; elements.<br/>In OpenDRIVE, lane width is described by the #lt;width#gt; element within the #lt;lane#gt; element. <br/>
